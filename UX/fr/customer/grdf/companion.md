# Animer une session d'activités web

Les activités web vous permettent d'inclure vos collaborateurs au cours de plusieurs types d'exercices
- Brainstorming
- Votes
- Questions fermées (J'aime / J'aime pas, Vrai / Faux et Oui / Non)
- Transferts de fichiers (Envoi, Réception)

Chaque utilisateur peut se connecter via application en ligne, reliée à votre session du Compositeur Digital UX.
Les sessions et toutes leurs données sont détruites dès que vous quittez votre environnement de présentation.

## Démarrer les activités web

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Démarrer les activités web**
4. Cliquer sur **Afficher lien web**
5. Un QR code apparaît, il peut être scanné par vos collaborateurs. L'url de la session est également visible sous le QR code.

## Démarrer un brainstorming

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Idées**
4. Une activité **Idées** est désormais manipulable dans l'espace de travail.
5. Cliquez sur le bouton **play** de cet élément.
6. Les utilisateurs connectés à la session peuvent désormais partager leurs idées. 
7. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
8. Toutes les idées reçues sont classées par utilisateur. Pour révéler les idées d'un utilisateur, cliquez sur son nom. Une feuille blanche, remplie de notes apparaît alors. Les notes peuvent ensuite être déplacées et triées. Il est également possible d'afficher toutes les notes en cliquant sur le bouton *Tous*.

## Démarrer une activité en mode anonyme

Par défaut, les activités utilisent l'identité des participants. Il est possible de basculer en mode "anonyme". L'exemple ci-dessous s'applique pour les activités **Idées**, mais le principe est le même pour toutes les activités.

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Idées**
4. Une activité **Idées** est désormais manipulable dans l'espace de travail.
5. Cliquer sur le bouton *...* 
6. Cliquer sur le bouton *Anonymat désactivé*. Il apparaît désormais comme sélectionné, et le label affiche à présent *Anonymat activé*. Il vous suffit de recliquer sur ce bouton pour sortir du mode anonymat. Attention, une fois l'activité démarrée dans ce mode, il n'est plus possible de le changer. Si vous faîtes ce changement, les activités que vous créérez par la suite seront toutes initiées en mode anonyme.  
7. Cliquer sur le bouton **play** de cet élément.
8. Les utilisateurs connectés à la session peuvent désormais partager leurs idées. 
9. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
10. Un seul élément apparaît, avec les idées de tout le monde, sans possibilité d'afficher l'auteur.

## Démarrer une activité avec un temps limite

Par défaut, les activités se lancent sans limite de temps. Vous êtes libre de choisir quand elles s'arrêtent. Il est néanmoins possible de les démarrer avec une durée limitée. L'exemple ci-dessous s'applique pour les activités **Idées**, mais le principe est le même pour toutes les activités.

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Idées**
4. Une activité *Idées** est désormais manipulable dans l'espace de travail.
5. Cliquer sur le bouton *chrono* 
6. Sélectionner une durée
7. Cliquer sur le bouton **play** de cet élément.
8. Les utilisateurs connectés à la session peuvent désormais partager leurs idées. 
9. A la fin du temps imparti, une minuterie retentie, et vous pouvez arrêter l'activité en cliquant sur **stop** ou la laisser continuer.

## Démarrer un vote basé sur des documents

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Votes**, puis **Vote documents**
4. Une activité *Vote documents* est désormais manipulable dans l'espace de travail
5. Choisissez les documents à soumettre au vote. Ceux-ci doivent être affichés sur l'espace de travail ou présents dans votre sélection. 
6. Cliquez sur le bouton **play** de cet élément
7. Les utilisateurs connectés à la session peuvent désormais faire leur choix. Tant que vous n'avez pas arrêté l'activité, les utilisateurs peuvent revenir sur leur choix.
8. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
9. Les trois premiers choix apparaîssent alors.

## Démarrer un vote dessin

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Votes**, puis **Vote dessin**
4. Une activité **Vote dessin** est désormais manipulable dans l'espace de travail
5. Vous disposez de 3 zones dans lesquelles il est possible de dessiner. Il n'est pas nécessaire de remplir toutes les zones pour lancer l'activité.
6. Cliquez sur le bouton **play** de cet élément quand les dessins sont prêts
7. Les utilisateurs connectés à la session peuvent désormais faire leur choix. Tant que vous n'avez pas arrêté l'activité, les utilisateurs peuvent revenir sur leur choix.
8. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
9. Les scores obtenus par chaque dessin apparaissent.

## Démarrer un vote note

Cette activité ne se base sur aucun contenu. Elle permet de recueillir rapidement une note, comprise entre 1 et 5, donnée par l'ensemble des participants.
Exprimez une idée à évaluer, présentez un document, puis démarrez l'activité.

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Votes**, puis **Note**
4. Une activité **Note** est désormais manipulable dans l'espace de travail 
5. Cliquez sur le bouton **play** de cet élément
6. Les utilisateurs connectés à la session peuvent désormais donner leur note.
7. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
8. La note moyenne s'affiche. Il est possible de détacher cette note. Pour cela cliquez sur le bouton *appareil photo*. Un sticker apparaît, vous pouvez le coller sur le contenu évalué.

## Démarrer une activité question

Toutes les activités **Questions** (J'aime / J'aime pas, Vrai / Faux et Oui /Non) ne se basent sur aucun contenu. Elles permettent de recueillir rapidement une réponse simple, donnée par l'ensemble des participants.
Exprimez une question fermée, un choix, un document, puis démarrez l'activité.

1. Ouvrir le menu principal
2. Cliquer sur le bouton *people*
3. Cliquer sur **Questions**, puis sélectionner le type de question.
4. Une activité **Question** est désormais manipulable dans l'espace de travail 
5. Cliquez sur le bouton **play** de cet élément
6. Les utilisateurs connectés à la session peuvent désormais donner leur réponse.
7. Pour arrêter l'activité, cliquez sur le bouton **stop** sur le bord.
8. Les détails des réponses s'affichent, avec le choix dominant. Il est possible de détacher le score. Pour cela cliquez sur le bouton *appareil photo*. Un sticker apparaît, vous pouvez le coller sur le contenu évalué.

## Démarrer une réception de fichiers

1. Ouvrir le menu principal
2. Cliquez sur le bouton *people*
3. Cliquer sur **Transferts**, puis **Réception**
4. Une activité est désormais manipulable dans l'espace de travail.
5. Cliquez sur le bouton **play** de cet élément.
6. Les utilisateurs connectés à la session peuvent désormais partager des fichiers. Tous les fichiers envoyés par les utilisateurs sont visibles dans l'activité, et également dans l'onglet **Importés** du menu principal.
7. Pour arrêter l'activité, cliquez sur le bouton **stop** en bas à droite

## Démarrer un envoi de fichiers

1. Ouvrir le menu principal
2. Cliquez sur le bouton *people*
3. Cliquer sur **Transferts**, puis **Envoi**
4. Une activité est désormais manipulable dans l'espace de travail.

## Démarrer une prise en main à distance

Il est possible de donner la main sur un contenu à un participant. Celui-ci pourra alors annoter le contenu ou bien naviguer dans le document et changer la page visible dans le Compositeur Digital.

1. Choisissez le document sur lequel le contrôle doit être donnée. 
2. Cliquez sur le bouton **people** de cet élément (il apparaît en bas à droite, au dessus du bouton **...**)
3. Choisissez le participant.
4. Une bordure de couleur apparaît alors autour du contenu, indiquant qu'il est manipulé par le participant.
5. Pour arrêter la prise en main, cliquez sur les initiales du participant en bas à droite, ou démarrer une autre activité.
